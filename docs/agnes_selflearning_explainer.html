First, let's clear something up
It's not what you think
"Self-learning AI" sounds complex. Agnes's approach is much simpler — and more transparent.
🧬
Common Misconception

"It retrains the AI model with new data"

People assume self-learning means the AI's internal weights or parameters change every time it's used. This would be slow, expensive, and opaque.

📚
What Agnes Actually Does

"It builds a library of past decisions and reads them before each new evaluation"

Agnes writes every decision to a memory file and reads the most relevant ones back into its prompt — like a doctor consulting case notes before seeing the next patient.

The core mechanism
A 3-step learning loop
Every evaluation runs these three steps automatically, growing smarter with each cycle.
①

Evaluate

Gemini AI analyses two ingredient suppliers and decides whether one can safely replace the other across all affected products.

gemini-flash-latest
②

Remember

The verdict — ingredient pair, grades, certifications, confidence score and full reasoning — is immediately written to a memory file.

KB/decisions.json
③

Recall

Next time, before calling Gemini, Agnes searches the memory file for similar past cases and shows them to the model as "Precedent Cases".

retrieve_similar_cases()

Safety by design
Humans stay in control
Agnes never saves a decision to memory without a human confirming it first.
🧑‍⚖️

The Human Confirmation Gate

When Agnes returns an evaluation in the Gradio UI, nothing is written to memory yet. The decision is only saved after a human explicitly chooses what to do with it:

✓ Apply — save this verdict ✗ Reject All — override and save as REJECT

This means the memory file only ever contains human-verified decisions — bad AI guesses can't silently corrupt future evaluations.

The big picture
Why this approach is powerful
Three key benefits for supply chain teams.
💸

Zero Retraining Cost

Agnes's AI model weights never change. Learning is stored in a plain JSON file — no GPU, no MLOps pipeline.

📈

Improves With Every Use

After 10 evaluations the system has 10 cases. After 1,000, it has institutional knowledge your team never had before.

🔍

Fully Auditable
